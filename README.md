# FPGA-Based MobileNetV2 Hardware Accelerator

## Description
Designed and implemented a custom RTL INT8 MAC using a Baugh-Wooley multiplier and Carry Lookahead Adder (CLA) in Verilog HDL for MobileNetV2 acceleration on FPGA.

## Tools
- Verilog HDL
- Vivado
- PYNQ-ZU FPGA
- Google colab

## Features
- INT8 MAC unit
- Baugh-Wooley signed multiplier
- Carry Lookahead Adder
- custom mobilenetv2 using numpy in python of pynq

## Files
- verilog_mac.v
- python_mobilenetv2.v
